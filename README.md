# PetShop - E-Commerce Platform for Pet Products and Adoptions

A full-stack web application for managing a pet shop business, featuring product sales and pet adoption services. The platform includes separate interfaces for customers and administrators, with secure authentication and role-based access control.

- **Product Management**: Create, update, and manage product inventory
- **Adoption Management**: Add and manage pet adoption listings
